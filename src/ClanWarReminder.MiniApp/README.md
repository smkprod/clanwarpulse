# ClanWarReminder Mini App

React + Vite + MUI frontend for Telegram Mini App.

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm install
npm run build
```

Build output goes directly to:

`../ClanWarReminder.Api/wwwroot/miniapp`

This allows ASP.NET API to serve the bundled frontend at `/miniapp`.
